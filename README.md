# Typeecho-MysqlDictionary
typeecho 文章解析数据库字典插件

## 使用方法

编写一篇文章，文章内容如下填写

```
<!--mysql-dictionary-->
<!--mysql-dictionary-host=127.0.0.1-->
<!--mysql-dictionary-username=root-->
<!--mysql-dictionary-password=root-->
<!--mysql-dictionary-database=testdb-->
<!--mysql-dictionary-port=3306-->
```

查看文章时,自动根据配置生成数据字典
